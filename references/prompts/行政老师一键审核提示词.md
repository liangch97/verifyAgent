# 行政老师一键审核提示词

把下面内容复制给 OpenClaw 使用。只需要替换尖括号里的内容。

## 默认审核合同（自动尝试企查查）

```text
请帮我审核这个合同，<把合同文件完整路径粘贴在这里>
```

示例：

```text
请帮我审核这个合同，<本项目目录>\samples\华为-中大智算集群可靠性测评技术合作协议-脱敏版.docx
```

默认会尝试复用本机企查查登录状态并核验甲方企业信息；如果登录状态失效，会弹出二维码；如果企查查核验无法完成，也要继续完成普通合同审核并说明企业核验未完成。

## 只做本地审核，不查企查查

```text
请帮我审核这个合同，但不要打开企查查，只做本地合同审核：<把合同文件完整路径粘贴在这里>
```

## 合同审核 + 已有企业信息材料

```text
请帮我审核这份横向合同，并根据我提供的材料核验甲方企业信息：

合同文件：<把合同文件路径粘贴在这里>
甲方企查查截图/页面文字/人工核验记录：<有就粘贴；没有就写“暂未提供”>
课题组成员：<把项目负责人和课题组成员姓名写在这里；没有就写“暂未提供”>

请不要自动打开企查查或其他企业信息网站。如果我没有提供企业核验材料，请继续完成合同审核，并把甲方企业信息列为需要补充企查查截图或人工核验记录的事项。

最终回复请面向行政老师，不要展示代码、命令、JSON、字段名、规则编号或系统内部过程。
```

## 合同审核 + 企查查扫码核验甲方

```text
请帮我审核这个合同，并通过企查查扫码登录核验甲方企业信息：<把合同文件完整路径粘贴在这里>

只查询合同里的甲方公司。遇到登录、验证码、访问限制或付费内容时不要绕过；如果无法形成可用的企业核验材料，请继续完成合同审核，并说明企业核验未完成。

如果 Dashboard 聊天框没有直接显示二维码截图，请返回“企查查截图预览页”的本地路径，我会打开预览页扫码或核对截图。

最终回复请面向行政老师，不要展示代码、命令、JSON、字段名、规则编号或系统内部过程。
```

如在受信任的本机上希望减少重复扫码，可补一句：

```text
这是受信任的本机，可以复用上次企查查扫码登录状态；如果登录状态已失效，再弹出二维码让我扫码。
```
